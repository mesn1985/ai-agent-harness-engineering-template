# Requirement Alignment Review

## Question

```text
Does the solution satisfy the stated intent, requirements, and acceptance criteria?
```

## Checklist

```text
- The implemented behavior matches the task objective.
- All stated acceptance criteria are addressed.
- Domain rules are respected.
- Edge cases are considered.
- Any assumptions are explicitly reported.
```

## Escalate If

```text
- Requirements are ambiguous.
- Acceptance criteria conflict.
- The implementation requires a product decision.
```
