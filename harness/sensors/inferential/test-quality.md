# Test Quality Review

## Question

```text
Do the tests verify meaningful behavior and important edge cases?
```

## Checklist

```text
- Tests cover the core behavior.
- Tests cover error cases.
- Tests avoid overfitting to implementation details.
- Regression tests are added for bug fixes.
- Test names describe behavior clearly.
```

## Escalate If

```text
- The expected behavior is unclear.
- The system lacks test infrastructure.
- A meaningful test requires unavailable dependencies.
```
