# Inferential Sensors

Inferential sensors require reasoning, interpretation, or judgment.

They answer questions such as:

```text
Is this the right solution?
Does this match the intent?
Is the design maintainable?
Are the tests meaningful?
What is the likely cause of this failure?
Is this risk acceptable?
```

Typical sensors:

- Requirement alignment review
- Architecture review
- Code maintainability review
- Test quality review
- Risk assessment
- Security judgment
- UX or domain judgment
- Human review
- AI critique
