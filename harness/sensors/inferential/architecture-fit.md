# Architecture Fit Review

## Question

```text
Does the solution fit the existing architecture, boundaries, and conventions?
```

## Checklist

```text
- Existing patterns are followed.
- Business logic is placed in the correct layer.
- Dependencies point in the expected direction.
- No unnecessary framework or dependency is introduced.
- The solution is maintainable and localized.
```

## Escalate If

```text
- A new architectural pattern is needed.
- Existing boundaries are unclear.
- The task requires broad refactoring.
```
