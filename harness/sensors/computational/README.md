# Computational Sensors

Computational sensors are executable, measurable, or rule-based checks.

They answer questions such as:

```text
Does it compile?
Do the tests pass?
Does the API contract hold?
Did the error rate increase?
Was the required file changed?
Does the output match the expected schema?
```

Typical sensors:

- Unit tests
- Integration tests
- End-to-end tests
- Type checks
- Linters
- Static analysis
- Security scans
- Build checks
- Contract tests
- Log queries
- Metrics checks
