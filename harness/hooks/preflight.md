# Preflight Hook

## Purpose

Prepare the agent before work begins.

## Command

```bash
./scripts/agent-preflight.sh
```

## Expected Use

The agent should run or mentally complete preflight before making changes.

## Preflight Checks

```text
- Read AGENTS.md.
- Read harness/feedforward.md.
- Read the task file.
- Identify risk level.
- Identify required sensors.
- Identify escalation boundaries.
```
