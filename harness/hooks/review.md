# Review Hook

## Purpose

Route completed work to a human or specialist reviewer.

## Required For

```text
- Medium, high, and critical risk tasks
- Public API changes
- Security-sensitive changes
- Architecture changes
- Data model changes
```

## Review Questions

```text
- Does the solution satisfy the requirement?
- Does it fit the architecture?
- Are tests meaningful?
- Are risks clearly reported?
- Are assumptions explicit?
```
